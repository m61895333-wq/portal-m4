# Portal M4

Portal editorial independente do Grupo M4, focado em mercado financeiro, investimentos, tecnologia, inteligencia artificial, carreira e negocios digitais.

## Estrutura

- `src/app`: paginas publicas, artigos, categorias e painel editorial.
- `src/lib`: camada de dados, Supabase, categorias, SEO e geracao editorial.
- `scripts/setup-db.mjs`: cria as tabelas do Supabase e insere posts iniciais.

## Deploy

Este projeto deve ser publicado como app independente na Vercel, sem depender da pasta antiga da SALEX.
